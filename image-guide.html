HTML Image Guide - Different Ways to Add Images

1. Basic Image Tag

The most common way to add images in HTML is using the <img> tag.

Basic Syntax:

<img src="path/to/image.jpg" alt="Description of image">

Example from your codebase:

<img src="assets/images/about/about.png" alt="About">

2. Images with Width and Height Attributes

You can specify the dimensions directly in the HTML attributes.

<img src="image.jpg" alt="Description" width="300" height="200">

Example from your codebase:

<img src="https://hannahgurlah.github.io/best-regards.png" alt="Hannah Pañares" height="40">

3. Images with CSS Styling

You can add inline CSS styles to customize the appearance.

<img src="image.jpg" alt="Description" style="border-radius: 10%; margin-bottom: 10px;">

Example from your codebase:

<img src="https://hannahgurlah.github.io/hannah-bw.jpg" alt="Hannah Pañares" width="120" style="border-radius: 10%;">

4. Images with CSS Classes

Using CSS classes for styling and animations.

<img class="animate-element delay7 fadeInUp" src="assets/images/about/about.png" alt="About">

Example from your codebase:

<img class="shape-one animate-element delay10 bounceIn" src="assets/images/about/shape-one.png" alt="Shape">

5. Responsive Images

Making images responsive for different screen sizes.

<img src="image.jpg" alt="Description" style="max-width: 100%; height: auto;">
<img src="image.jpg" alt="Description" class="img-fluid">

6. Images with Links

Making images clickable by wrapping them in anchor tags.

<a href="https://example.com"> <img src="image.jpg" alt="Clickable image"> </a>

7. Images with Figure and Figcaption

Using semantic HTML for images with captions.

<figure> <img src="image.jpg" alt="Description"> <figcaption>Caption for the image</figcaption> </figure>

8. Background Images with CSS

Using CSS to set images as background.

<div style="background-image: url('image.jpg'); background-size: cover; height: 300px;"> Content here </div>

9. Images with Loading Attributes

Optimizing image loading performance.

<img src="image.jpg" alt="Description" loading="lazy">
<img src="image.jpg" alt="Description" loading="eager">

10. Images with Srcset for Different Resolutions

Providing different image sizes for different screen densities.

<img src="image.jpg" srcset="image-small.jpg 300w, image-medium.jpg 600w, image-large.jpg 900w" sizes="(max-width: 600px) 300px, (max-width: 900px) 600px, 900px" alt="Responsive image">

11. Images with Picture Element

Using the picture element for more control over image sources.

<picture> <source media="(min-width: 800px)" srcset="large-image.jpg"> <source media="(min-width: 400px)" srcset="medium-image.jpg"> <img src="small-image.jpg" alt="Fallback image"> </picture>

12. Images with Object-fit

Controlling how images fit within their containers.

<img src="image.jpg" alt="Description" style="object-fit: cover; width: 300px; height: 200px;">
<img src="image.jpg" alt="Description" style="object-fit: contain; width: 300px; height: 200px;">

Important Attributes for Images:

  • src: The path to the image file (required)
  • alt: Alternative text for accessibility (required for accessibility)
  • width: Width of the image in pixels
  • height: Height of the image in pixels
  • loading: "lazy" or "eager" for loading behavior
  • srcset: Multiple image sources for different screen sizes
  • sizes: Sizes attribute for responsive images

Best Practices:

  • Always include the alt attribute for accessibility
  • Use appropriate image formats (JPEG for photos, PNG for graphics with transparency)
  • Optimize images for web (compress file sizes)
  • Use responsive images for better performance
  • Consider lazy loading for images below the fold
  • Provide fallback images when using modern techniques
